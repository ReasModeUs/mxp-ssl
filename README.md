# MRZ SSL Manager

Automated SSL Certificate Manager for VPS (Marzban, Sanaei, PasarGuard).

## Features
- Auto-detect and free Port 80
- Auto-deploy to Marzban
- User-friendly path display for other panels
- English interface & Logs

Run the following command on your server:
